About This Demo

Purpose

This is a generic demonstration project showcasing a modern testing framework using Python pytest and Playwright. It demonstrates both UI testing and API testing capabilities in a simple, easy-to-understand format.

Technology Stack

  • Backend: FastAPI - Modern Python web framework
  • Frontend: Static HTML, CSS, and JavaScript
  • UI Testing: Playwright with pytest
  • API Testing: pytest with requests library
  • CI/CD: GitHub Actions

Testing Coverage

  • ✓ Navigation and page routing tests
  • ✓ Form validation and submission tests
  • ✓ Dynamic content loading tests
  • ✓ REST API endpoint tests (GET, POST, DELETE)
  • ✓ Health check and error handling tests

API Endpoints

GET /api/health - Health check
GET /api/items - List all items
POST /api/items - Create a new item
GET /api/items/{id} - Get item by ID
DELETE /api/items/{id} - Delete item by ID

Project Structure

Tests are organized in separate directories: ui_tests/ for Playwright browser tests and api_tests/ for REST API tests. This structure makes it easy to run specific test suites independently.
